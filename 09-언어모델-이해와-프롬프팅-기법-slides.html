📋 AI 직원에게 업무 지시하는 3가지 방법

신입사원에게 업무 매뉴얼을 주는 것처럼 AI에게도 명확한 지시가 필요합니다
📖 업무 지시서 (시스템 메시지)
AI 직원의 기본 설정:
  • 역할과 업무 스타일 정의
  • 답변 톤과 주의사항
  • 전체적인 업무 환경 설명
예시: "당신은 고객 서비스 전문가입니다. 친절하고 정확한 답변을 제공해주세요."
💼 실제 업무 요청 (사용자 메시지)
구체적인 업무 내용:
  • 구체적인 질문이나 업무
  • 처리할 자료나 데이터
  • 이전 단계 정보 활용
예시: "고객이 보낸 다음 문의에 답변해주세요: '배송이 지연되고 있어요'"
⚙️ 세부 조정 (옵션 설정)
AI 성격 세밀 조정:
  • 창의성: 0(보수적) ~ 1(창의적)
  • 답변 길이: 최대 글자 수 제한
  • 다양성: 답변 변화 정도
설정 예시: 창의성 70%, 답변 500자 이내, 다양성 90%

📝 완벽한 업무 매뉴얼 작성하는 4단계

신입사원 교육처럼 AI에게도 체계적인 업무 가이드가 필요합니다
🎯 체계적인 매뉴얼 구조
  1. 역할 정의
    "당신은 누구인가요?"
  2. 행동 지침
    "어떻게 일해야 하나요?"
  3. 출력 형식
    "어떤 형식으로 답변하나요?"
  4. 주의사항
    "절대 하면 안 되는 것은?"
💼 실제 예시: 고객 상담 전문가
1단계 - 역할 정의: 당신은 우리 회사의 전문 고객 상담원입니다. 2단계 - 행동 지침: - 항상 친절하고 정중하게 대해주세요 - 고객의 마음을 이해하고 공감해주세요 - 확실하지 않은 것은 "확인 후 연락드릴게요" 3단계 - 답변 형식: 1. 고객 문의 내용 간단히 정리 2. 고객 감정 상태 파악 3. 해결 방법 제시 4. 추가 도움 확인 4단계 - 주의사항: - 개인정보는 절대 물어보지 마세요 - 답변은 500자 이내로 간단명료하게

🎓 AI 직원 교육 고급 기법

전문적인 AI 직원 교육을 위한 4가지 핵심 기법을 마스터하세요
📊 Format Specification (구조화된 출력)
JSON 스키마를 활용한 정확한 데이터 형식 지정
고객 피드백 분석 출력 형식: ```json { "sentiment": "긍정/중립/부정", "category": "제품/서비스/배송/기타", "priority": "높음/중간/낮음", "key_issues": ["이슈1", "이슈2"], "suggested_actions": ["액션1", "액션2"], "follow_up_required": true/false } ``` 위 JSON 형식으로만 답변해주세요.
📚 Teaching by Examples (예시 기반 학습)
실제 모범 사례를 보여주며 패턴 학습 유도
고객 피드백 분석 예시: 예시 1: 입력: "배송이 너무 늦어서 실망했습니다" 출력: { "sentiment": "부정", "category": "배송", "priority": "중간", "key_issues": ["배송지연"], "suggested_actions": ["배송상태 확인", "사과 메일 발송"] } 이제 다음을 분석하세요: {{ $json.feedback }}
📋 Spec Files (프로젝트 명세서)
spec.md 파일로 프로젝트 정보와 가이드라인 전달
spec.md 파일 구조: # 프로젝트 명세서 ## 목표 - 고객 만족도 향상 - 응답 시간 단축 ## 브랜드 가이드라인 - 톤: 친근하고 전문적 - 스타일: 간결하고 명확 ## 제약사항 - 개인정보 수집 금지 - 500자 이내 답변 이 spec을 참고하여 작업해주세요.
🧠 Chain of Thoughts (단계별 사고)
planner.md 파일로 문제 해결 과정을 체계화
planner.md 파일 예시: # 고객 문의 처리 플래너 ## 단계별 분석 과정 1. 문의 내용 파악 및 분류 2. 고객 감정 상태 분석 3. 우선순위 결정 4. 해결방안 도출 5. 적절한 응답 생성 각 단계를 순서대로 수행하며 사고 과정을 명시해주세요.

Step 2: Chain of Thoughts 분석
1. 불만 유형: 제품 불량 + 감정적 대응 필요 2. 긴급도: 높음 (즉시 환불 요구) 3. 고객 감정: 분노 + 실망 4. 해결책: 즉시 환불 + 추가 보상 5. 브랜드 회복: 진정성 있는 사과
