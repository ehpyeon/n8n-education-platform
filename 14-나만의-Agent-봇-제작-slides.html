🚀 뼈대 구축 단계

Step 1: Webhook 설정

  1. Webhook Trigger 노드 추가
  2. HTTP Method: POST 선택
  3. Path: /agent-bot 설정
  4. Webhook URL 복사 (나중에 사용)

Step 2: Agent 노드 연결

  1. AI Agent 노드 추가
  2. Webhook에서 Agent로 연결
  3. System Prompt 작성
  4. Tools 섹션 준비 (아직 비어있음)

Step 3: LLM & Memory 설정

  • LLM: Upstage Chat Model 선택
  • Model: solar-pro2 선택
  • Memory: Window Buffer Memory 추가
  • Window Size: 5 설정 (최근 5개 대화)

Step 4: Sub-Workflow 호출 준비

  • Tools 섹션에서 "Call n8n Workflow" 선택
  • 4개 Tool 슬롯 준비 (다음 단계에서 연결)
  • 각 Tool 이름과 설명 플레이스홀더 작성
  • 입력/출력 스키마 정의
