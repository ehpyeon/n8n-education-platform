📋 AI 직원에게 업무 지시하는 3가지 방법

신입사원에게 업무 매뉴얼을 주는 것처럼 AI에게도 명확한 지시가 필요합니다
📖 업무 지시서 (시스템 메시지)
AI 직원의 기본 설정:
  • 역할과 업무 스타일 정의
  • 답변 톤과 주의사항
  • 전체적인 업무 환경 설명
예시: "당신은 고객 서비스 전문가입니다. 친절하고 정확한 답변을 제공해주세요."
💼 실제 업무 요청 (사용자 메시지)
구체적인 업무 내용:
  • 구체적인 질문이나 업무
  • 처리할 자료나 데이터
  • 이전 단계 정보 활용
예시: "고객이 보낸 다음 문의에 답변해주세요: '배송이 지연되고 있어요'"
⚙️ 세부 조정 (옵션 설정)
AI 성격 세밀 조정:
  • 창의성: 0(보수적) ~ 1(창의적)
  • 답변 길이: 최대 글자 수 제한
  • 다양성: 답변 변화 정도
설정 예시: 창의성 70%, 답변 500자 이내, 다양성 90%

📝 완벽한 업무 매뉴얼 작성하는 4단계

신입사원 교육처럼 AI에게도 체계적인 업무 가이드가 필요합니다
🎯 체계적인 매뉴얼 구조
  1. 역할 정의
    "당신은 누구인가요?"
  2. 행동 지침
    "어떻게 일해야 하나요?"
  3. 출력 형식
    "어떤 형식으로 답변하나요?"
  4. 주의사항
    "절대 하면 안 되는 것은?"
💼 실제 예시: 고객 상담 전문가
1단계 - 역할 정의: 당신은 우리 회사의 전문 고객 상담원입니다. 2단계 - 행동 지침: - 항상 친절하고 정중하게 대해주세요 - 고객의 마음을 이해하고 공감해주세요 - 확실하지 않은 것은 "확인 후 연락드릴게요" 3단계 - 답변 형식: 1. 고객 문의 내용 간단히 정리 2. 고객 감정 상태 파악 3. 해결 방법 제시 4. 추가 도움 확인 4단계 - 주의사항: - 개인정보는 절대 물어보지 마세요 - 답변은 500자 이내로 간단명료하게

🎓 AI 직원 교육 고급 기법

모범 답안을 보여주면서 가르치는 방법으로 AI 성능을 크게 향상시킬 수 있습니다
📚 예시로 가르치기 (Few-shot Learning)
교육 방법:
업무 매뉴얼 + 모범 답변 예시들 + 실제 업무 요청
고객 문의 분류 교육 예시:
  • "배송이 늦어요" → 물류팀
  • "결제가 안 됩니다" → 결제팀
  • "제품 사용법을 모르겠어요" → 기술지원팀
이런 예시 몇 개만 보여주면 새로운 문의도 정확하게 분류!
🔄 실전 교육 프로세스
고객 문의 이메일 자동 분류 교육: 예시 1: 이메일: "주문한 제품이 언제 배송되나요?" 분류: 문의유형=배송, 긴급도=중간, 감정=중립 예시 2: 이메일: "제품이 불량품이에요! 즉시 환불해주세요!" 분류: 문의유형=환불, 긴급도=높음, 감정=분노 예시 3: 이메일: "서비스가 정말 만족스럽습니다." 분류: 문의유형=피드백, 긴급도=낮음, 감정=긍정 이제 다음 이메일을 분류해주세요: "{{ $json.email_content }}"

🧠 고급 프롬프트 전략 마스터

복잡한 업무도 단계별로 나누어 처리하면 AI가 더 정확하게 수행할 수 있습니다
🔍 Chain of Thought (사고 연쇄)
복잡한 문제를 단계별로 나누어 해결
비즈니스 분석 예시: 다음 단계별로 분석해주세요: 1단계: 현재 상황 파악 2단계: 문제점 식별 3단계: 원인 분석 4단계: 해결방안 도출 5단계: 실행계획 수립 분석 대상: {{ $json.business_data }} 각 단계별로 사고 과정을 명확히 보여주고 최종 결론을 제시해주세요.
🎭 Role-based Prompting (역할 기반)
특정 전문가 역할을 부여하여 전문적 답변 유도
마케팅 전문가 역할 예시: 당신은 10년 경력의 디지털 마케팅 전문가입니다. 전문 분야: - 소셜미디어 마케팅 - 콘텐츠 마케팅 - 데이터 기반 캠페인 최적화 다음 마케팅 데이터를 전문가 관점에서 분석: {{ $json.marketing_data }} 분석 항목: 1. 캠페인 성과 평가 2. 타겟 오디언스 분석 3. 개선 기회 식별 4. 구체적 실행방안 제시
