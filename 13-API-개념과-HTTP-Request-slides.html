  • 기존 문서 처리 워크플로우들 활용

핵심: 웹훅으로 외부 요청을 받고, 서브 워크플로우로 내부 처리를 연결하여 통합적인 자동화 시스템을 구축할 수 있습니다.

📞 웹훅(Webhook): 외부와의 통신 창구

웹훅은 외부에서 우리 n8n 워크플로우로 요청을 보낼 수 있게 해주는 특별한 주소입니다.

🏢 웹훅 = 회사의 대표 전화번호

  • 외부에서 접근 가능한 고유 URL 주소
  • 24시간 언제든 요청을 받을 수 있음
  • 받은 요청을 워크플로우로 전달
  • 처리 결과를 다시 응답으로 반환
웹훅 URL 예시: https://myworkflow.n8n.io/webhook/chat 외부 → 웹훅 → 워크플로우 → 응답

🎯 웹훅 활용 시나리오

상황 외부 요청 처리 결과
챗봇 연결 "문서 분석해줘" AI 분석 결과 응답
폼 제출 처리 고객 문의 데이터 자동 분류 및 알림
모바일 앱 연동 파일 업로드 요청 처리 상태 알림

⚙️ 웹훅 설정과 간단한 프론트엔드 연결

웹훅을 설정하고 간단한 웹페이지와 연결하여 외부 통신을 구현해보겠습니다.

🔧 n8n 웹훅 노드 설정

  • HTTP Method: POST (데이터 전송용)
  • Path: /chat (사용자 정의 경로)
  • Response Mode: Respond to Webhook
  • Authentication: None (테스트용)
웹훅 설정 결과: https://your-n8n.com/webhook/chat → 이 주소로 POST 요청하면 워크플로우 시작

🌐 간단한 HTML 프론트엔드

다음 단계: 이 웹훅이 받은 요청을 내부 워크플로우들과 연결하여 복잡한 처리를 수행할 수 있습니다.

🔄 서브 워크플로우: 내부 워크플로우 간 통신

서브 워크플로우는 기존에 만들어둔 워크플로우들을 다른 워크플로우에서 호출하여 사용하는 방법입니다.

🧩 모듈화의 장점

  • 기존 문서 처리 워크플로우 재활용
  • 복잡한 로직을 단계별로 분리
  • 각 워크플로우의 독립적 관리
  • 디버깅과 유지보수 용이성
메인 워크플로우: 웹훅 → 데이터 검증 → 서브 워크플로우 호출 서브 워크플로우: 문서 분석 → AI 처리 → 결과 반환

🎯 서브 워크플로우 활용 패턴

용도 서브 워크플로우 결과
문서 분석 PDF 텍스트 추출 구조화된 텍스트
AI 처리 내용 요약 및 분류 분석된 정보
알림 발송 결과 전송 사용자 알림

🔗 MCP 노드: 더 스마트한 워크플로우 연결

MCP(Model Context Protocol) 노드는 워크플로우 간의 더 지능적이고 유연한 연결을 가능하게 합니다.

🧠 MCP 노드의 특징

  • 컨텍스트 유지 가능
  • 지능적 데이터 전달
  • 오류 처리 자동화
  • 동적 워크플로우 호출

🔄 서브 워크플로우 vs MCP

구분 서브 워크플로우 MCP 노드
연결 방식 고정적 동적
데이터 전달 단순 지능적
오류 처리 수동 자동

🏗️ 외부-내부 통합 시스템 구조

웹훅과 서브 워크플로우를 조합하여 완전한 통합 자동화 시스템을 구축할 수 있습니다.

🌊 전체 데이터 흐름

외부 요청 (웹페이지/앱) ↓ 웹훅 (메인 워크플로우) ↓ 요청 검증 및 라우팅 ↓ 서브 워크플로우 호출 ↓ - 문서 처리 워크플로우 - AI 분석 워크플로우 - 알림 발송 워크플로우 ↓ 결과 통합 및 응답 ↓ 외부로 결과 반환

⚡ 시스템의 장점

  • 확장성: 새로운 기능을 서브 워크플로우로 쉽게 추가
  • 재사용성: 기존 워크플로우들을 조합하여 활용
  • 유지보수: 각 부분을 독립적으로 수정 가능
  • 실시간 처리: 외부 요청을 즉시 처리
예시: 문서 분석 봇 웹훅 → 파일 검증 → 업스테이지 분석 → AI 요약 → 결과 저장 → 사용자 알림

💼 실제 구현 예시: 문서 분석 통합 시스템

웹훅과 기존 문서 처리 워크플로우들을 연결한 실제 구현 사례를 살펴보겠습니다.

🎯 메인 워크플로우 구성

  1. 웹훅 노드
    • 경로: /document-analysis
    • 메소드: POST
    • 파일 업로드 받기

    • 파일 형식 검증 (PDF, DOCX)
    • 파일 크기 제한 확인
  3. 서브 워크플로우 호출
    • 문서 파싱 워크플로우
    • AI 분석 워크플로우
