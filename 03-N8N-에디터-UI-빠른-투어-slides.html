🖥️ N8N 에디터 UI 빠른 투어

작업실 둘러보기 - 어디에 뭐가 있는지 알아보자

  1. 왼쪽 메뉴 - 내 작업실의 정리된 서랍장
  2. 중앙 캔버스 - 자동화를 그림 그리듯 만드는 작업판
  3. 노드 패널 - 다양한 업무 도구들의 보관함
  4. 첫 자동화 - "Hello World" 워크플로우 만들기

🏢 작업 공간 둘러보기

정리정돈된 작업실의 서랍장처럼 필요한 것들이 체계적으로 배치되어 있습니다
메뉴 역할 일상 비유
📊 Overview 모든 자동화 시스템 한눈에 보기 작업실 전체를 한눈에 보는 감시실
👤 Personal 내 개인 작업 공간 내 개인 책상과 서랍
📁 Projects 프로젝트별로 정리된 공간 프로젝트별로 나눈 파일함

✨ 첫 번째 자동화 시작하기

새 워드 문서를 만드는 것처럼 쉽게 첫 워크플로우를 생성합니다
  1. 새 자동화 만들기
    • Overview 클릭 후 Create Workflow 버튼 클릭
    • 새 워드 문서 만들기와 동일한 개념
  2. 이름 지어주기
    • "첫 번째 자동화" 같은 알아볼 수 있는 이름으로 저장
    • 단축키: Ctrl + S (윈도우) / Cmd + S (맥)
  3. 템플릿 활용하기
    • 다른 사람이 만든 자동화를 복사해서 내 것으로 만들기
    • Workflows → Duplicate로 복제 (요리 레시피 복사하기)
  4. 내 상황에 맞게 수정하기
    • 복제한 자동화의 설정을 내 회사/상황에 맞게 변경
    • History 탭에서 이전 버전 확인 및 복원 가능

🎨 메인 캔버스 (작업 영역)

실제 워크플로우를 그려나가는 디지털 화이트보드입니다

🔧 캔버스 구성 요소

  • 점선 격자 배경 - 워크플로우 편집 공간
  • "Add first step" 점선 사각형 - 첫 번째 노드 추가 위치
  • Execute workflow 버튼 - 모든 노드 순차적 실행
  • + 아이콘 버튼 - 노드 패널 열기
  • 노트 아이콘 - 캔버스에 스티키 노트 추가
  • Ask Assistant 버튼 - AI 워크플로우 구축 도움

🎮 캔버스 제어 기능

  • 줌 조절 - 화면에 맞게 줌 조절
  • 줌 인/아웃 - 세부 편집 및 전체 보기
  • 줌 리셋 - 기본 크기로 복원
  • 화면 정리 - 노드 자동 정렬

🖱️ 캔버스 네비게이션

  • Ctrl + 좌클릭으로 드래그 이동
  • 마우스 중간 버튼으로 드래그 이동
  • 터치패드에서 두 손가락으로 슬라이드

🧩 노드 패널 (우측)

레고 블록을 선택하는 것처럼 필요한 기능 노드를 골라 사용합니다

🔑 노드 패널 열기 방법

  • 캔버스 우상단 + 아이콘 클릭
  • 기존 노드 우측 + 아이콘 클릭
  • 키보드 Tab 키 누르기

📦 노드 분류

  • 첫 번째 노드 - 다양한 트리거 노드 카테고리
  • Advanced AI - AI 기능 노드들
  • Actions in an App - 앱 연동 액션
  • Data transformation - 데이터 변환
  • Flow - 흐름 제어
  • Core - 핵심 기능
  • Human in the loop - 사람 개입 기능

⚙️ 노드 관리 기능

각 노드를 효율적으로 제어하고 편집하는 방법을 익힙니다

🎛️ 노드 제어 (마우스 호버 시)

  • Execute step - 해당 노드만 실행
  • Deactivate - 노드 비활성화
  • Delete - 노드 삭제
  • Node context menu - 노드 액션 메뉴

✏️ 노드 이름 변경

  • 노드 선택 후 F2 키
  • 노드 더블클릭 → 좌상단 이름 클릭 → Rename
  • 노드 우클릭 → Rename 옵션 선택

▶️ 실행 관리

워크플로우를 테스트하고 모니터링하는 핵심 기능입니다

📊 실행 로그 접근 방법

특정 워크플로우 실행:

  • 워크플로우 캔버스에서 Executions 탭 선택
  • Editor 탭으로 노드 에디터 복귀

전체 실행 로그:

  • 에디터 UI에서 Overview 선택
  • 메인 패널의 Executions 탭 선택

🔄 워크플로우 활성화

  • 상단 내비게이션의 Inactive 토글을 Activated로 설정
  • 우상단 점 3개 메뉴에서 Settings 접근

⚙️ 실행 설정 옵션

워크플로우 실행을 최적화하고 문제를 추적하는 설정들입니다

⏱️ 타임아웃 설정

  • EXECUTIONS_TIMEOUT - 최대 실행 시간(초) 설정
  • 무한 실행 방지 및 리소스 소비 제어

💾 실행 데이터 저장

  • Save failed executions - 실패한 실행 데이터 저장 (기본: 저장)
  • Save successful executions - 성공한 실행 데이터 저장 (기본: 저장)
  • Save manual executions - 수동 실행 데이터 저장 (기본: 저장)
  • Save execution progress - 각 노드 실행 데이터 저장 (에러 지점 재개 가능, 기본: 저장 안 함)

🔐 크레덴셜 관리

외부 서비스와 안전하게 연결하기 위한 인증 정보를 관리합니다

🔑 크레덴셜 개요

  • 앱과 서비스 인증을 위한 개인 정보
  • n8n과 외부 서비스 간 연결 브리지
  • 암호화되어 안전하게 저장됨

📁 인터페이스 접근

  • 좌측 메뉴에서 "Credentials" 선택
  • My credentials - 내가 생성한 크레덴셜
  • All credentials - 공유된 크레덴셜 포함

🛡️ 보안 기능

  • 암호화 저장 - 민감한 정보 보호
  • OAuth 인증 - "connect" 버튼으로 간편 로그인
  • 필드 숨김 - 민감한 정보 자동 숨김
  • 접근 제어 - 노드별 권한 관리

🚀 효율적인 워크플로우 개발 순서

UI 요소들을 조합하여 체계적으로 자동화를 구축하는 방법입니다
  1. 좌측 패널에서 프로젝트 선택
    • Overview 또는 Personal 선택
    • Create Workflow 클릭하여 새 워크플로우 시작
  2. 노드 패널에서 트리거 선택
    • Tab 키 또는 + 아이콘 클릭으로 노드 패널 열기
    • 트리거 노드 카테고리에서 적절한 노드 선택
  3. 노드 설정 및 크레덴셜 연결
    • 노드 더블클릭으로 설정 패널 열기
    • 필요한 크레덴셜 선택 또는 새로 생성
  4. 워크플로우 확장 및 연결
    • 노드 우측 + 아이콘으로 다음 노드 추가
    • Advanced AI, Actions, Data transformation 등에서 선택
  5. 테스트 및 활성화
    • Execute workflow 또는 개별 노드 Execute step으로 테스트
    • 상단 토글을 Activated로 설정하여 자동 실행 활성화
